# save-code

## Description

This is a simple project to save code snippets.

## Technologies

- electron
- vite
- vue3
- tailwindcss
- pinia
- bytemd
- antdv
- vue-router
- vue-codemirror 
- pinia-plugin-persistedstate

## Installation

```bash
npm install 
npm run dev
npm run build:win or npm run build:mac
```
